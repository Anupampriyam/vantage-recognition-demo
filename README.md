# Vantage Recognition — Interactive Demo

A self-contained, embeddable interactive demo of the Vantage Recognition platform
(guided iPad walkthrough: recognize a teammate, see it in the feed, redeem points).

Live: served via GitHub Pages (see repo Pages settings).

## Embed on any page

```html
<div style="position:relative;width:100%;max-width:1080px;margin:0 auto;padding-top:74%;">
  <iframe src="https://<pages-url>/index.html"
          style="position:absolute;inset:0;width:100%;height:100%;border:0;"
          title="Vantage Recognition demo" loading="lazy" allowtransparency="true"></iframe>
</div>
```
